About Basic or Bizarre


Basic or Bizarre is an open source tool that calculates a uniqueness score for a playlist based on the popularity of it's tracks. Created by Prachi Heda.

How is the popularity calculated?

The popularity of each playlist is calculated by averaging the popularity score of each track in the playlist. The popularity score is an index created by Spotify that ranks tracks based on total streams, playing history, and playing frequency.

What link do I enter?

Go to any public (not private) playlist on Spotify, click the three dots, Share, Copy link to playlist. It should look like https://open.spotify.com/playlist/(some playlist ID)

Why isn't the website working?

If you've already logged in, try logging out and logging in again. If incorrect information is shown after calculating, click calculate again. If this does not work, refresh or force refresh your browser (Shift+cmd+R). Lastly, make sure the playlist link you are entering is public.
